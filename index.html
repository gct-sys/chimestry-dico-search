À Propos de Ce Bloc

Ce code ne contient plus les balises de page complète. Il est optimisé pour être intégré dans un cadre (iframe) de Google Sites.

Comment ça marche

  • **HTML :** Le contenu ci-dessous est affiché.
  • **CSS :** Le style dans la balise **<style>** s'applique uniquement aux éléments à l'intérieur de ce bloc (classe .custom-widget).

Ceci remplace votre pied de page précédent.
